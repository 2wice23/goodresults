AI-Powered Site Features

These tools use AI through Netlify proxy functions and work 24/7 for anyone on the team, no computer required:

Call Analyzer — Upload a call recording, get scored and coached by AI

Deal Generator — Pull property data and generate deal packages with AI

Deal Doc Builder — AI generates formatted deal documentation from inputs

These call the Anthropic API via anthropic-proxy.js on Netlify. The API key lives in the AnthropicAPI env var. No Cowork session needed.

How This Works

Server tasks are Netlify scheduled functions that run on cron schedules, 24/7. They handle monitoring, digests, and simple data pulls. They do not need AI — just HTTP calls to GitHub, Slack, and the module API.

AI tasks run in Cowork (Claude desktop app) and require an active session on Gayden's computer. These handle things that need intelligence — analyzing call data, merging knowledge bases, writing coaching reports, composing Slack messages with context. If the computer is off or Cowork isn't running, these tasks pause and pick up at the next session.

The site tools (Analyzer, Deal Generator, Deal Docs) use AI but run through Netlify proxy functions, so they work for the whole team around the clock.
