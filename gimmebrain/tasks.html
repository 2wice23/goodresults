Automations
What This System Does
🎓
Academy — 12 Training Modules
Full onboarding curriculum covering pipeline, cold calling, offers, contracts, financing, due diligence, foreclosures, SMS playbooks, and more. Each module has a quiz pool that tracks which questions you have already mastered — once you prove you know it, you stop seeing it. Notes box on every module lets agents flag mistakes or suggest content, and Gayden reviews them daily.
🎙
Call Analyzer — AI Call Scoring
Paste any call transcript and AI scores it across 7 criteria using the Good Results playbook — buy box delivery, objection handling, tone, forbidden language, closing attempt, and follow-up. Shows exactly what you missed and the script you should have used at each point. Every analyzed call feeds back into the knowledge base, so scoring improves over time. The leaderboard tracks daily, weekly, and all-time averages per agent.
🧠
Self-Learning Knowledge Base
Five interconnected KBs (Master, Analyzer, Chloe Voice Agent, SMS, Academy) that sync automatically. New call data gets pulled daily from the Google Sheet, compared against existing knowledge, and proposed as updates — nothing changes without Gayden's approval. Sunday master merges push new knowledge down to every small brain. The system literally gets smarter every week without anyone writing documentation.
🤖
Chloe — AI Voice Agent
AI-powered phone agent in Close CRM that handles inbound and outbound calls using the same playbook the human agents train on. Chloe's knowledge base stays in sync with the master KB through the weekly merge cycle, so when the team learns something new, Chloe learns it too.
🚀
Deal Launcher — One-Click Deal Setup
Upload a signed contract and property photos, fill in deal details, hit Launch. It creates the deal page on the website, updates the opportunity in Close CRM, and generates a full disposition workflow with AI-written texts and emails from the SMS playbook. Replaces 45 minutes of manual setup across three systems with one button.
🔍
KB Search — Cross-Brain Search
Search every knowledge base at once — Academy modules, analyzer playbook, Chloe scripts, and SMS cadences. Results are color-coded by source so you know where the info lives. If your search comes up short, the question box at the bottom sends it to the Slack Q&A channel and tags Joe and Gayden automatically.
📊
Deal Dashboard — Live Pipeline View
Real-time view of every active deal across Las Vegas, Reno, and Houston. Pipeline status, contract dates, EMD deadlines, and disposition progress. Check it every morning to know what needs attention — which deals are approaching deadlines, which dispositions need follow-up, and where the pipeline stands across all three markets.
⚡
Automated Daily Operations
8 scheduled automations run without any computer being on — morning briefings, hourly call target tracking with agent tagging, weekly coaching reports, KB learning cycles, and master merges. Slack gets the right message in the right channel at the right time. Leadership is shown on scoreboards but never tagged. Everything that needs approval pauses and waits.
🔒
Secrets Proxy Architecture
Zero API keys in the source code. Every external API call (Close CRM, Slack, OpenAI, GitHub) goes through Netlify serverless proxy functions with keys stored in environment variables. The repo is public and safe to share — no credentials are exposed anywhere in the codebase.
🏗
Auto-Deploy Pipeline
Push to GitHub, Netlify builds and deploys automatically. No FTP, no manual uploads, no drag-and-drop. Every deployment is version-controlled with full commit history. One standardized push method (git clone/push) across every project and scheduled task — learned the hard way after two sessions using different methods overwrote each other.
