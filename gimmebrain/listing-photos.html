Claude Tool

Listing Photo
Downloader

Paste a Zillow, Realtor.com, or Redfin listing URL into Claude — and every property photo gets downloaded to a named folder in seconds. No more right-click-saving 40 photos one by one.

How It Works
1
Share the URL
Paste any Zillow, Realtor.com, or Redfin listing link into Claude in Cowork.
2
Auto-Extraction
Claude navigates the listing, opens the gallery, and pulls every photo at max resolution from the page data.
3
Folder Ready
Photos download in parallel to a folder named after the property address. Numbered and verified.
Supported Sites
Zillow
Full resolution (1536px)
Realtor.com
Full resolution via CDN
Redfin
Full resolution via CDN
What You Get
Output Details
📸
Every photo — not just the first few. Pulls from the page data so nothing gets missed, even if the gallery is lazy-loaded.
🔍
Highest resolution available — deduplicates by image hash and keeps the largest version (1536px on Zillow).
📁
Named folder — auto-creates a folder from the property address. Example: 3035 Everett Dr Reno NV
🔢
Numbered files — photo_01.jpg, photo_02.jpg, etc. Clean and sorted.
⚡
Parallel downloads — 10 images at a time. A 40-photo listing takes seconds, not minutes.
✅
Verified — checks file count, validates JPEG data, flags any failures. You’ll know immediately if something didn’t work.
How to Use
⚡ Quick Start
1
Open Claude in Cowork (this runs through the Claude desktop app).
2
Paste the listing URL and say something like grab the pics from this listing or download photos from this.
3
Claude opens the listing in Chrome, extracts all photo URLs, and downloads them to your workspace folder.
4
You’ll see a summary — total photos, folder name, and sample images to confirm everything pulled correctly.
